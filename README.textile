h1. ■フレームワーク

役割分担。
１つのことをうまく処理する小さなコードが協調し合って、全体の仕事を成しとげる。

h2. ■MVCアーキテクチャ

* Model      ... データ出し入れ専門
* View       ... HTML描画専門
* Controller ... 各所に処理を指示する役目

h2. ■処理の流れ

# すべてのリクエストを一点に集める（.htaccessへリライト記述しエントリスクリプトindex.php等に集める）
# URI分析し、リクエストされたアクション（コントローラのメソッド）に処理を渡す
# コントローラからモデルへデータ出し入れ指示
# コントローラからビューへHTML作成を指示し、完成されたHTMLを返却してもらう
# コントローラから、HTMLレスポンス
